# 恋爱依赖度测试网站

一个专为20-38岁女性设计的恋爱幸福度测试网站，采用AI智能分析，提供个性化的评语和建议。

## 功能特点

✨ **核心功能**
- 激活码验证系统（一次性使用）
- 20道精心设计的测试题目
- 5个维度评分（激情度、沟通质量、付出程度、幸福感、恋爱阶段）
- AI智能评语生成（根据分数区间个性化定制）
- 流式输出评语（打字机效果）

🎨 **设计特色**
- 粉色主题，温暖治愈
- 玫瑰色渐变背景
- 柔和的动画效果
- 响应式设计，支持移动端

## 如何使用

### 1. 生成激活码

访问以下URL生成激活码（需要管理员密码）：

```
http://你的域名/api/activation/generate?count=10&password=love2025admin
```

参数说明：
- `count`: 生成数量（1-100）
- `password`: 管理员密码（默认：love2025admin）

返回示例：
```json
{
  "success": true,
  "message": "成功生成 10 个激活码",
  "codes": [
    "LOVE-BNDH-DG3P-4UXH",
    "LOVE-WRP9-REXZ-2AMV",
    ...
  ]
}
```

### 2. 开始测试

1. 访问网站首页
2. 输入激活码（格式：LOVE-XXXX-XXXX-XXXX）
3. 点击"开始测试"按钮
4. 完成20道测试题目
5. 查看测试结果和AI评语

### 3. 分享结果

在结果页面点击"分享给朋友"按钮，可以将测试链接分享给好友。

## 测试说明

### 恋爱阶段分类
- **热恋期**：1-3个月
- **平淡期**：3-6个月
- **磨合期**：6-12个月
- **适应期**：12-28个月
- **婚恋期**：28个月以上

### 评分维度
- **激情度**：衡量感情中的激情和浪漫程度
- **沟通质量**：衡量双方的沟通和理解程度
- **付出程度**：衡量双方为这段感情的付出和努力
- **幸福感**：衡量在这段感情中的整体幸福程度
- **恋爱阶段**：当前所处的恋爱阶段

### 分数区间评语风格
- **0-20分**：犀利但带同情，指出问题所在
- **21-50分**：中肯客观，给予具体建议
- **51-60分**：温和鼓励，肯定美好之处
- **61-75分**：温柔说好话，提醒爱惜自己
- **76-90分**：热情夸奖，天时地利人和
- **91-100分**：充满祝福，一定会幸福

## 注意事项

1. **激活码一次性使用**：每个激活码只能使用一次，使用后立即失效
2. **测试结果仅供参考**：测试结果由AI生成，仅供参考
3. **隐私保护**：测试数据不会存储，保护用户隐私
4. **管理员密码**：建议修改默认管理员密码（在 `src/app/api/activation/generate/route.ts` 中修改）
